# study-mcp-host-with-gradio

## .env有効化

`.env.example`をコピーして`.env`ファイルを作成し、必要な環境変数を設定してください。

```bash
cp .env.example .env
```

`.env`ファイルを開いて、`ANTHROPIC_API_KEY`に実際のAPIキーを設定してください。

```env
ANTHROPIC_API_KEY=your_actual_api_key_here
```
